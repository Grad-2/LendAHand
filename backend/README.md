# API Routes:

## GET api/
> ### Request Example:
> > No Request Parameters
> ### Response Example:
> > ```
> > {
> >     "active": true
> > }
> > ```

> >     {
> >         "id": 1,
> >         "username": "testuser1"
> >     },
> >     {
> >         "id": 2,
> >         "username": "testuser2"
> >     }
> > ]
> > ```
